O Que é HTML: Desvendando a Linguagem de Marcação para a Web

(Voltar)

o HTML, é a sigla para "HyperText Markup Language" (Linguagem de Marcação de Hipertexto, em português). O HTML é uma linguagem de marcação fundamental para o desenvolvimento de páginas web. Ele permite a estruturação e organização de conteúdo, rebaixando o navegador como exibindo diferentes elementos, como textos, imagens, links e outros recursos.

HTML utiliza "tags" para envolver e definir os diferentes elementos em uma página. Cada tag tem uma função específica, como criar parágrafos, títulos, listas, entre outros. Combinado com CSS (Cascading Style Sheets) e JavaScript, o HTML desempenha um papel crucial na construção e design de páginas web interativas e visualmente atraentes.

O que são Tags

(Voltar)

Uma "tag" no HTML é uma parte fundamental da linguagem de marcação que define a estrutura e o significado dos elementos em uma página web. As tags são utilizadas para marcar o início e o fim dos elementos HTML, indicando como o navegador deve interpretar e exibir determinado conteúdo.

As tags geralmente são compostas por um nome entre colchetes angulares, como < tag> e < /tag >. A tag de abertura < tag >sinaliza o início de um elemento, enquanto a tag de fechamento indica o final desse elemento.

Por exemplo, uma tag < p > é usada para criar um parágrafo. O texto que está entre < p > e < /p > será interpretado como um parágrafo pelo navegador. Veja um exemplo:
